### Infinite Mac: Active Development Continues; Blog Post Activity June 2026


Infinite Mac allows users to pick any version of System Software, Mac OS, Mac OS X, or NeXTStep from the 1980s, 1990s, or early 2000s and run it within a virtual machine, with an "Infinite HD" disk containing representative software from each era.
 
The project runs Basilisk II, SheepShaver, Mini vMac, DingusPPC, Previous, and PearPC as its underlying emulator cores, hosted in separate repositories as Git submodules.
 
A comment was posted on the project's original 2022 launch blog post as recently as June 3, 2026,
 indicating continued community engagement. The project's GitHub releases page shows ongoing commits.

**Source:** [Infinite Mac — infinitemac.org](https://infinitemac.org/) | [GitHub](https://github.com/mihaip/infinite-mac)

## Hardware

### LisaFPGA Project: Open-Source FPGA Apple Lisa — YouTube Video Released This Week


User AlexTheCat123 announced on the 68kMLA forum (dated May 3, 2026) that after roughly eight months of work, a fully functional Apple Lisa implementation inside an FPGA is nearly complete, including a custom board with all Lisa connectors, HDMI audio/video output, USB keyboard/mouse input, onboard ESProFile-based hard disk emulation, and overclocking support up to approximately 4× the Lisa's stock clock speed.
 
A companion YouTube video titled "How I Recreated the Apple Lisa Inside an FPGA — The (Open-Source) LisaFPGA Project" was published June 3, 2026,
 drawing wider community attention this week. 
The board is centered around an Artix 7-100T FPGA coupled to 2 MB of SRAM with emulated hard drive, floppy, serial, and input connectors.
 
Pricing is variable by order volume, ranging from approximately $500 per board for small runs down to $80 per board for larger quantities.


**Source:** [The Apple Lisa Inside an FPGA! — 68kMLA](https://68kmla.org/bb/threads/the-apple-lisa-inside-an-fpga.52207/) | [Hackaday coverage](https://hackaday.com/2026/05/09/its-an-apple-lisa-on-a-fpga/) | [YouTube](https://www.youtube.com/watch?v=2QSP9Qh0PvU)

### TashTalk USB: Active Forum Thread, Community Testing in Progress


A community member has developed a USB-to-TashTalk board based on a CP2102N and a full-duplex transceiver, designed to work with or without a LocalTalk dongle, with the stated goal of making it straightforward to transfer files between any classic Mac with a LocalTalk port and a modern Mac/Linux/Windows machine.
 
Schematics and board files are published on GitHub, and a GUI for controlling the server is described as nearly ready for Linux and Mac user trials.
 The 68kMLA forum thread for TashTalk USB 
was receiving replies as of this week (June 8, 2026),
 with community members reporting practical results. 
One user reported getting SE/30s and an LC475 communicating with each other and with a MacBook Pro via the TashTalk USB device and PhoneNet.


**Source:** [TashTalk USB — 68kMLA](https://68kmla.org/bb/threads/tashtalk-usb.52115/)

### SE/30 Simasimac Troubleshooting — Active Thread This Week


A current 68kMLA thread active as of this week documents ongoing troubleshooting of a Macintosh SE/30 exhibiting a simasimac display pattern combined with a boot failure, with community guidance noting that receiving any chime indicates the machine is likely loading ROM and a serial cable can be used to read error codes in this condition.
 
The thread references the Simasimac Repair Guide's recommendation to test continuity between the PDS slot and ROM slot rather than the CPU directly for ease of access, while also noting that broken traces between the CPU and ROM may not appear in PDS-only tests.
 The thread exemplifies the ongoing volume of SE/30 repair activity on the forum.

**Source:** [Yet another simasimac SE/30 giving me grief — 68kMLA](https://68kmla.org/bb/threads/yet-another-simasimac-se-30-giving-me-grief.52361/)

### DMCA Triennial Rulemaking — Game Preservation Comment Period Opens June 2026


The next DMCA triennial rulemaking is expected to begin in June 2026, with its outcome expected to shape what is legally possible for game preservation through the end of the decade.
