### Infinite Mac: Continued Community Reach

Infinite Mac is a browser-based emulator that allows users to run every version of classic MacOS from System 1.0 to MacOS 9.2.2, and even transfer files between their modern computer and the emulator.
 
This is made possible through WebAssembly — JavaScript and HTML5 still power a majority of the web, but when low-level, high-performance capabilities are needed, WebAssembly provides a powerful tool running inside modern web browsers.
 No new version release was announced this week, but community discussion on 68kMLA referencing the project remains active.

**Source:** [infinitemac.org](https://infinitemac.org) · [Low End Mac coverage](https://lowendmac.com/2026/infinitemac-run-classic-mac-os-in-your-browser/)

## Gaming

### *Innecto* — Modern-to-Vintage Reverse Port (Ongoing thread active this week)

Developer VMSZealot ported their game *Innecto* to the Mac SE as a personal platform of choice. The port was described as unfinished at announcement, with requests not to upload it to the Garden yet. Testing was conducted on an LC475 (System 7.6), a Mac SE (System 6.0.8), and a Mac 512ke (System 2.1).
 
The developer described the project as exploring the minimum specification Innecto will run on — noting it runs on an M4 Mac, an Apple TV, a Watch, and has now been confirmed to run on a Mac 512, potentially even an original Mac 128.
 
The associated 68kMLA thread "Innecto: A new game for System 7 (68000)" received a reply as recently as Friday, June 13, 2026.
