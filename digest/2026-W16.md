### Infinite Mac — Browser-Based Classic Mac Emulation Suite

Infinite Mac is a collection of classic Macintosh and NeXT system releases accessible from a web browser, covering any version of System Software, Mac OS, Mac OS X, or NeXTStep from the 1980s through early 2000s, with an "Infinite HD" disk containing representative era software.
 
Users can run a custom version with their own machine choice and disk images, and on some operating systems can import/export files and mount virtual CD-ROMs via drag and drop.
 The project's changelog ("what's changed recently") remains accessible from the main page.

**Source:** [Infinite Mac](https://infinitemac.org/)

### *Reckless Drivin'* SDL Port — Updated Build on Mac Source Ports

Mac Source Ports is distributing version v1.0.4 of the *Reckless Drivin'* SDL port, built March 19, 2026, as a Universal 2 binary for Apple Silicon and 64-bit Intel Macs, with the full game included and no additional data required.
 
The original *Reckless Drivin'* is a top-down driving game released in 2000 by Jonas Echterhoff, who open-sourced it in 2019; the modern port was developed by Darren Cohen.
 
The original source relies heavily on deprecated Macintosh system calls, with game data stored as LZRW3-compressed assets inside the resource fork of the original game binary.


**Source:** [Mac Source Ports – Reckless Drivin'](https://www.macsourceports.com/game/recklessdrivin/)

## Hardware

### Phoenix — Open-Source NuBus FPGA Accelerator for 68040 Macs

User Sailcat posted a design for an open-source NuBus FPGA accelerator card targeting the Quadra 700 and other 68040 NuBus machines, aimed at providing hardware acceleration for TLS 1.3 cryptography, hardware blitting, basic video decode, DSP, and general-purpose compute without replacing Classic Mac OS.
 
A KiCad schematic, design document, and BOM are ready for community review prior to PCB layout; the project uses a Lattice ECP5 LFE5U-85F FPGA chosen specifically for its open-source toolchain (Yosys + nextpnr + Project Trellis).
 
Estimated BOM cost is $60–90 at quantity 1, with total power draw around 2–3W, within the NuBus per-slot budget.
 The thread remains active with replies as recently as this weekend (April 12, 2026).

**Source:** [68kMLA – Phoenix NuBus FPGA thread](https://68kmla.org/bb/threads/phoenix-open-source-nubus-fpga-accelerator-for-68040-macs.51658/)

### EBD_IPKVM — Raspberry Pi Pico-Based Classic Mac Video Capture and KVM

User h4rm0n1c presented EBD_IPKVM, a Raspberry Pi Pico-based framebuffer using firmware and PIO programs to capture digital video signals from vintage Macs spanning the Lisa through the SE/30.
 
As of February 10, 2026, keyboard and mouse input were confirmed working, with the system running a Raspberry Pi Pico and an Arduino handling video, keyboard, and mouse over two USB connections.
 
The approach uses a Raspberry Pi Pico with standard jellybean DIP parts and voltage regulation, without requiring oversampling or dedicated CPLD hardware, opening the possibility for exact 1:1 format video capture comparable to what Basilisk II or Mini vMac display.
 The thread has received continued replies into this week.

**Source:** [68kMLA – EBD_IPKVM thread](https://68kmla.org/bb/threads/ebd_ipkvm-an-all-digital-approach-to-classic-mac-video-capture-and-control.51553/) | [GitHub: h4rm0n1c/EBD_IPKVM](https://github.com/h4rm0n1c/EBD_IPKVM)

## Community Projects

### MARCHintosh 2026 — GlobalTalk AppleTalk Networking Event

GlobalTalk is a community practice of using Apple Internet Router software to bridge local AppleTalk networks to the global internet, enabling vintage Mac users to share files over AppleShare and print to each other's AppleTalk-enabled printers connected over LocalTalk/PhoneNet cables; usage primarily spikes in March for MARCHintosh.
 
Updated guides and pre-made disk images to simplify GlobalTalk onboarding were published March 1, 2026.
 
One contributor added LToUDP bridge support to a fork of jrouter (a modern Apple Internet Router), enabling GlobalTalk participation and bridging to a real ImageWriter via TashTalk with no retro hardware required.
 TinkerDifferent's MARCHintosh 2026 banner remains active on the forum homepage.
