## Emulation

### Infinite Mac — Web-Based Classic Mac Emulator Continues Active Use


Infinite Mac allows users to pick any version of System Software or Mac OS from the 1980s through early 2000s and run it in a virtual machine, with an "Infinite HD" disk pre-loaded with representative software from each era.
 
The project integrates Basilisk II, SheepShaver, Mini vMac, DingusPPC, Previous, and PearPC as its underlying 68K and PowerPC emulator cores, hosted as Git submodules.
 No new release was published this week, but the project continues to draw discussion across the community. 
Community members have noted that the project allows running a virtualized 68K Mac in the browser with Macintosh Garden disc files available for one-click loading, though performance drops on NeXTStep and Mac OS X presets while classic macOS systems run well.


**Source:** [infinitemac.org](https://infinitemac.org) | [GitHub: mihaip/infinite-mac](https://github.com/mihaip/infinite-mac)

## Hardware

### IIsiA7 Mini — FPGA-Based HDMI Adapter for the Macintosh IIsi PDS Slot (Active Thread)


The IIsiA7 Mini is a smaller sibling of the IIsiFPGA, described as the "Highly Desirable Macintosh Interface for the IIsi," sharing a GitHub repository with the original version and self-identifying as IIsiFPGA V3.0.
 
In this revision, the FPGA is soldered directly on the board rather than on a daughterboard, reducing thickness.
 
The board supports multiple HDMI PHY options: a DVI-compatible mode with hardware resolutions from 640×480 to 1920×1080 at various bit depths with windowboxing but no audio, and a full HDMI mode at 1920×1080 with windowboxing and audio.
 
A known bug in the gateware causes burst reads from memory to fail due to a width converter issue introduced by the DDR2 1:2 ratio; memory expansion mode functions but at reduced speed.
 The thread received replies as recently as this week.

**Source:** [68kMLA thread — IIsiA7 Mini](https://68kmla.org/bb/threads/iisia7-mini-hdmi-for-the-68030-pds-slot-bis.51956/)

### MacDVI — Open-Source Pico 2 TTL-to-DVI Video Converter for Mac Plus/128k/512k


A forum member announced the publication of MacDVI, a DIY project designed to move a Mac Plus to an LCD display, with source code and documentation hosted on Codeberg at arroz/MacDVI.
 
The project uses a single Raspberry Pi Pico 2 to sample the Mac's TTL video signal and outputs to a 1024×768 DVI (HDMI-compatible) display; because that resolution is exactly twice the Mac's native horizontal resolution, the result is pixel-perfect scaling with black bars compensating for the 384×342 height difference.
 
It has been confirmed to work on the 128k, 512k, and Plus; other black-and-white compact Macs may be compatible with software adjustments.
 
The author states that extensive documentation was written with the intent for the project to serve as learning material for those getting into electronics or compact Mac restoration.
 The thread was indexed approximately two weeks ago and appeared in 68kMLA's active discussions this week.

**Source:** [68kMLA thread — MacDVI](https://68kmla.org/bb/threads/macdvi-oss-pico-2-based-ttl-to-dvi-video-converter.52268/) | [Codeberg: arroz/MacDVI](https://codeberg.org/arroz/MacDVI)

## Community Projects

### CompleteSet — G3–G5 Mac Accessory Reference Resource Receiving Ongoing Updates


User mac27 announced CompleteSet, an online resource for vintage Mac collectors targeting the G3–G5 era (roughly 1998–2006), with the goal of providing a complete, accurate reference for determining the accessories, peripherals, software disks, and documentation that originally shipped with vintage Apple systems.
 
The project addresses the historically difficult determination of which exact materials shipped with specific systems, a task previously requiring educated guesswork and partial information from EveryMac.
 
The third and final category — "Documentation" — went live, described as the most difficult portion to compile and still requiring additional contributions.
 The 68kMLA thread for this project received activity as recently as today (May 25, 2026).

**Source:** [68kMLA thread — CompleteSet](https://68kmla.org/bb/threads/completeset-a-resource-for-vintage-mac-collectors-and-enthusiasts.51006/) | [mac27.net/completeset](https://www.mac27.net/completeset)
