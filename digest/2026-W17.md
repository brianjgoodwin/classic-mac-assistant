## Emulation

### DingusPPC: Active Commits Through April 19–20

DingusPPC, the experimental PowerPC Mac emulator under active development, received multiple commits in the days immediately preceding this digest. 
A mirror of the DingusPPC repository on Applefritter Git was synced as recently as April 20, 2026, with a commit by Maxim Poliakovski on April 19 improving DMA transfer handling in the `sc53c94` SCSI controller.
 
A separate commit dated April 1, 2026 switched the `scsicdrom` device to use generic `ScsiCdromCmds`, reducing the file size significantly (188 deletions).
 
DingusPPC is a Power Mac emulator for Windows, Linux, and macOS that aims to emulate actual hardware rather than patching ROM and RAM as SheepShaver does; the most functional machines are the PowerMac 6100 and PowerMac G3 Beige, and the emulator is described as under active development and updated frequently.

### Infinite Mac: Macintosh Garden Library Integration & Comment Activity


Infinite Mac allows users to pick any version of System Software, Mac OS, Mac OS X, or NeXTStep from the 1980s, 1990s, or early 2000s and run it within a virtual machine in the browser; an "Infinite HD" disk with representative software from each era is also available, and the project supports custom machines and disk choices for embedding.
 
A comment was posted to the project's Macintosh Garden Library blog entry as recently as April 11, 2026.
 
The Macintosh Garden Library integration gives the project an archive of nearly every piece of software released in the 1980s and 1990s; any item in the Garden's catalog of 20,000 applications and games can be loaded into the emulated Mac with one click, based on a JSON dump of the catalog provided by the Garden's maintainer.


**Source:** [Infinite Mac – Macintosh Garden Library post](https://blog.persistent.info/2024/11/infinite-mac-macintosh-garden-library.html)

### QemuMac: Bash Toolkit for Classic Mac QEMU Setup


QemuMac is a bash toolkit for classic Mac emulation via QEMU, providing pre-configured VMs for a Quadra 800 (68K) and a PowerMac G4 (PPC) running Mac OS 7.5–9.2 and OS X Tiger/Leopard, with auto-downloads of ROMs and installers and a curated retro software catalog.
 
A user announced the toolkit on E-Maculation, describing `run68k.sh` as a launcher using simple `.conf` files to define QEMU settings — machine type, RAM, ROM, disk images, PRAM, and graphics — allowing easy switching between different Mac OS installs without constructing long command lines each time; disk and PRAM images are automatically created if they do not exist.


**Source:** [QemuMac on GitHub](https://github.com/matthewdeaves/QemuMac) | [E-Maculation thread](https://www.emaculation.com/forum/viewtopic.php?t=12455)

## Hardware

### IIsiA7 Mini: FPGA-Based HDMI Adapter for the 68030 PDS Slot


A 68kMLA forum member (Melkhior) posted the IIsiA7 Mini, a smaller sibling of the IIsiFPGA project that provides HDMI output for the Macintosh IIsi via its 68030 PDS slot, sharing a GitHub repository with the original version and designated internally as IIsiFPGA V3.0.
 
The new version has the FPGA soldered directly to the board rather than using a daughterboard, making it smaller (fitting within the IIsi adapter form factor) and reducing overall cost to less than two-thirds of the cost of the ZTex 2.12b daughterboard alone, though the PCB itself is a 6-layer board with a 256-ball Artix-7 FPGA.
 
The board supports multiple HDMI PHY options including a DVI-like mode with hardware resolutions from 640×480 to 1920×1080 at 1–32 bpp with windowboxing but no audio, and an HDMI mode at 1920×1080 with windowboxing and audio.
 
The author notes an active bug where burst reads from memory do not work despite working on the original IIsiFPGA — attributed to a width conversion issue between the DDR3 1:4 ratio on the original and DDR2 1:2 ratio on the new design — meaning memory expansion mode works but runs slower than expected.


The thread was posted March 29, 2026 and received replies as recently as this week per the 68kMLA "What's New" listing.

**Source:** [68kMLA – IIsiA7 Mini thread](https://68kmla.org/bb/threads/iisia7-mini-hdmi-for-the-68030-pds-slot-bis.51956/)
