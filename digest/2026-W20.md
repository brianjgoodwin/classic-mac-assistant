### QemuMac: QEMU Bash Toolkit for Classic Mac VMs


The `QemuMac` project on GitHub is a Bash toolkit for classic Mac emulation via QEMU, providing pre-configured VMs for a Quadra 800 (68k) and a PowerMac G4 (PPC) running Mac OS 7.5 through 9.2 and OS X Tiger/Leopard.
 
A recent update added support for launching PPC machines, added a setup script to install the latest QEMU for m68k and PPC from source, and introduced a `mac-library.sh` script that lets users pick from a list of CD images from Macintosh Garden and Archive.org for download and mounting.


**Source:** [matthewdeaves/QemuMac — GitHub](https://github.com/matthewdeaves/QemuMac) | [E-Maculation thread](https://www.emaculation.com/forum/viewtopic.php?t=12455)

### Infinite Mac: Browser-Based Classic Mac Emulation


Infinite Mac is a collection of classic Macintosh and NeXT system releases and software accessible from a web browser, covering versions of System Software, Mac OS, Mac OS X, and NeXTStep from the 1980s, 1990s, and early 2000s.
 
An "Infinite HD" disk with representative era software is available, and users can run custom instances with their choice of machine and disks, embed them into other sites, and on some operating systems import/export files and mount virtual CD-ROMs via drag and drop.
 The project page notes a changelog of recent updates at `infinitemac.org`.

**Source:** [Infinite Mac](https://infinitemac.org)

### DingusPPC and PearPC WebAssembly: PowerPC Mac OS in the Browser


Programmers have been working on DingusPPC, an emulator capable of running in WebAssembly in a web browser and able to boot Mac OS 9.2.2 and early Mac OS X releases. In parallel, PearPC has also been ported to WebAssembly. As of early 2025, both can boot Mac OS X 10.2 in the browser, though on a modern computer it takes nearly five minutes and runs too slowly for practical use.


**Source:** [Macintosh Repository](https://www.macintoshrepository.org/)

## Hardware

### IIsiA7 Mini: FPGA HDMI Board for the 68030 PDS Slot — Thread Active This Week


A new board called the IIsiA7 Mini has been presented on 68kMLA as the smaller sibling of the IIsiFPGA, designed to bring HDMI output to the Macintosh IIsi via the 68030 PDS slot. It shares a GitHub repository with the original version and is referred to internally as IIsiFPGA V3.0.
 
Unlike its predecessor, the IIsiA7 Mini has the FPGA soldered directly on the board with no daughterboard required, making it smaller and reducing overall cost to less than two-thirds the cost of the ZTex 2.12b module alone.
 
The board supports multiple HDMI PHY options, offering resolutions from 640×480 to 1920×1080 at 1/2/4/8/16/32 bpp with windowboxing, plus an optional audio-capable HDMI mode at 1920×1080.
 
The author notes a current bug where burst reads from memory do not work correctly, with a width converter required on the new DDR2 interface; memory expansion mode functions but runs slower than expected.
 The 68kMLA thread was active as of Sunday, May 10, 2026.

**Source:** [IIsiA7 Mini: HDMI for the 68030 PDS slot (bis) — 68kMLA](https://68kmla.org/bb/threads/iisia7-mini-hdmi-for-the-68030-pds-slot-bis.51956/)

## Gaming

### *Return to Dark Castle* — Steam Release Accumulating Reviews


*Return to Dark Castle*, released on Steam on March 3, 2026 by Z Sculpt Entertainment and Ludit, is a re-imagined retro platformer featuring all 30 levels from the original *Dark Castle* and *Beyond Dark Castle* plus 50 brand-new rooms, rebuilt for modern systems.
 
As of this week, recent reviews rate the game "Very Positive" at 91% of 12 recent reviews, with an overall "Very Positive" score of 94% across 68 total reviews.
 
The game was selected by Apple for the "Games We Love" feature on the Mac App Store.
 
The publisher, Ludit, was founded by Mark Stephen Pierce, co-creator of the original Dark Castle series, and is dedicated to reviving the franchise.


**Source:** [Return to Dark Castle — Steam](https://store.steampowered.com/app/4054940/Return_to_Dark_Castle/)

### Mac Source Ports: *Unreal* (1998) Port — March 2026


The OldUnreal team, which previously produced the Mac port of Unreal Tournament, received permission from Epic to release a build of the original *Unreal* (1998) for the Mac.
 
Although the original game was available for classic Mac OS, this constitutes the first Mac port in many years.
 
The original renderer has been replaced with a new Metal-based renderer; some glitches are expected but the game is described as playable.
 The build (v227k_14) was dated March 1, 2026.

**Source:** [Mac Source Ports: Unreal](https://www.macsourceports.com/game/unreal/)
