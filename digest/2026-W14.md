### Infinite Mac: Continued Attention to Browser-Based Classic Mac Emulation


Infinite Mac allows users to pick any version of System Software, Mac OS, Mac OS X, or NeXTStep from the 1980s through the early 2000s and run it within a virtual machine, with an "Infinite HD" disk of era-appropriate software available.
 
A prior significant update added early Mac OS X support, with versions 10.1 and 10.3 described as best supported, and rebuilt the Infinite HD library to include notable indie software from that period.
 
Infinite Mac uses WebAssembly builds of Basilisk II, SheepShaver, Mini vMac, DingusPPC, Previous, and PearPC, maintained as Git submodules.
 The project continued to receive public mentions in Low End Mac's community feed as recently as January 2026.

**Source:** [infinitemac.org](https://infinitemac.org) | [GitHub — mihaip/infinite-mac](https://github.com/mihaip/infinite-mac)

## Hardware

### IIsiA7 Mini — FPGA-Based HDMI Card for the Mac IIsi 68030 PDS Slot


User Melkhior posted the IIsiA7 Mini to 68kMLA on March 29, 2026 — a smaller sibling of the IIsiFPGA, described as an HDMI interface for the Mac IIsi's 68030 PDS slot, sharing a GitHub repository with the original and designated internally as IIsiFPGA V3.0.
 
The new board eliminates the daughterboard by soldering the FPGA (a 256-ball Artix-7) directly to a 6-layer PCB, reducing total solution cost to less than two-thirds of a ZTex 2.12b module alone.
 
It supports resolutions from 640×480 up to 1920×1080 at 1/2/4/8/16/32 bits per pixel, with an optional HDMI mode adding audio and windowing support.
 
A reported bug in the gateware prevents burst reads from memory from functioning as expected, slowing memory expansion mode below its target speed.


**Source:** [68kMLA — IIsiA7 Mini thread](https://68kmla.org/bb/threads/iisia7-mini-hdmi-for-the-68030-pds-slot-bis.51956/)
