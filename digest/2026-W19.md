### DingusPPC Alpha Releases: Disk Setup and Quake Support


A recent DingusPPC alpha release added the ability to format disks using the hard disk setup program, eliminating the need for tweaks or third-party programs.
 
A prior pre-release (Alpha 1.04) carried a note that the emulator can now run Quake.
 
The most recent alpha notes that SCSI emulation has been reworked, though further work remains, and that audio crash bugs, Mesh controller bugs, and implementation of a cached interpreter are outstanding before the next version milestone.


**Source:** [DingusPPC Releases — GitHub](https://github.com/dingusdev/dingusppc/releases)

### Infinite Mac: Browser-Based Classic Mac Emulation Continues Operation


Infinite Mac is a collection of classic Macintosh and NeXT system releases and software, accessible from a web browser, allowing users to pick any version of System Software, Mac OS, Mac OS X, or NeXTStep from the 1980s, 1990s, or early 2000s and run it within a virtual machine.
 
The project uses Basilisk II, SheepShaver, Mini vMac, DingusPPC, Previous, and PearPC as its underlying emulation engines, hosted as Git submodules.
 A Low End Mac community post from January 2026 noted renewed discussion of the project, with one user describing the ability to 
"run a virtualized 68K Mac IN BROWSER with some system disks, Macintosh Garden disc files already setup for one-click loading."


**Source:** [Infinite Mac](https://infinitemac.org/) | [Low End Mac — January 2026](https://lowendmac.com/2026/infinitemac-run-classic-mac-os-in-your-browser/)

### *Dark Castle* (1986) Legacy: Context on the Revival


The original *Dark Castle*, released in 1986, appeared on Macworld's Top Five best-seller list for more than 36 consecutive months, frequently claiming the No. 1 spot.
 
The 2026 release plays through the complete *Dark Castle* and *Beyond Dark Castle* rebuilt with updated visuals, remastered audio, and refined controls, followed by 50 new levels designed to push player skill further — described as a game of precision, timing, and persistence.
 
Mark Pierce, posting directly on AppleInsider forums, stated this release is the first step in reviving the Dark Castle series, and that a new entry is already in development.

## Hardware

### 68kMLA: PRAM Battery Damage and Radius 81/110 Issues Surface in Recent Posts


The 68kMLA forum's hardware sections show recent discussion of topics including a blue PRAM battery leak and questions about a Radius 81/110 system, with related posts visible in the current feed.
 These are recurring maintenance concerns in the vintage Mac community, as Maxell-brand PRAM batteries installed in compact Macs are known to cause logic board corrosion when they fail.

**Source:** [68kMLA Forums](https://68kmla.org/bb/)

---

### Macintosh Classic Reloaded: Logic Board Reproduction Project Background


The Macintosh Classic Reloaded project, hosted on TinkerDifferent, addresses machines that have died because of Apple's decision to add a battery to the board — specifically a battery prone to explosion — by producing a near 1:1 reproduction of the logic board.
 
The GitHub repository has been made private permanently due to eBay sellers stripping attributions and violating the CC-NC license; anyone wishing to purchase a board can do so through MacEffects.
 This project remains a reference point for community members dealing with battery-damaged Classics.

**Source:** [TinkerDifferent — Macintosh Classic Reloaded](https://tinkerdifferent.com/threads/logic-board-macintosh-classic-reloaded.26/)
