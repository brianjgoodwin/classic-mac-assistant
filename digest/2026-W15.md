## Emulation

### QEMU for macOS (PPC) — Experimental Builds Continue in February 2026


An experimental build of QEMU-system-ppc for macOS (ARM only, built on M1 with macOS Tahoe) was posted on February 23, 2026. The build changes the emulated PowerMac model to PowerMac3,3, adds improved SMP support running Mac OS 9.x through OS X 10.3 with 2 CPUs (and OS X 10.4/10.5 with 4 CPUs), adds improved USB audio support, and adds support for the Sungem/GMAC network device.



An earlier build from February 21 improved shared folder functionality for Mac OS 9.x, and a February 18 build added USB audio support and fixed aliases and programs copied to and from the shared folder.
 All builds are maintained by community moderator Cat_7 on the E-Maculation forums.

### DingusPPC — Browser-Accessible PowerPC Emulation Milestone


DingusPPC, a PowerPC Mac emulator with WebAssembly support, can now boot Mac OS 9.2.2 and early Mac OS X releases in-browser. In parallel, PearPC has also been ported to WebAssembly, and as of early 2025, both emulators can boot Mac OS X 10.2 in a browser, though the process takes nearly 5 minutes on modern hardware with limited practical usability.



DingusPPC aims to improve upon the accuracy of SheepShaver, PearPC, and other PowerPC Mac emulators, and is designed to work best with PowerPC NuBus and Old World ROMs.
 
Community forum discussion in November 2024 noted that DingusPPC now fully boots Mac OS 7.1 through 10.2, and the G3 Beige emulation can boot cleanly into Mac OS 8.6 and all versions of Mac OS 9, as well as back to Mac OS 8.1.


**Source:** [DingusPPC GitHub](https://github.com/dingusdev/dingusppc) | [Macintosh Repository](https://www.macintoshrepository.org/) | [E-Maculation QEMU/DingusPPC thread](https://www.emaculation.com/forum/viewtopic.php?t=12336)

### Infinite Mac — In-Browser Classic Mac Access


Infinite Mac is an active project providing in-browser access to classic Macintosh and NeXT system releases, covering System Software, Mac OS, Mac OS X, and NeXTStep from the 1980s through the early 2000s. An "Infinite HD" disk with representative era software is available, and users can build custom instances with their choice of machine and disk images, with drag-and-drop file import/export and virtual CD-ROM mounting supported on some operating systems.



DingusPPC's WebAssembly build is accessible through Infinite Mac under Custom:Experimental, where users can play with different hardware configurations and OS versions.


**Source:** [Infinite Mac](https://infinitemac.org/)

### GlobalTalk — AppleTalk Networking Infrastructure Updated for 2026


GlobalTalk is the community practice of using Apple's Apple Internet Router software to bridge a local AppleTalk network to the global internet, connecting vintage computers and printers across geographic boundaries. Usage primarily spikes during March for MARCHintosh, and allows participants to share files over AppleShare and print to each other's AppleTalk-enabled printers over LocalTalk/PhoneNet cables.



The MARCHintosh/GlobalTalk site posted a March 1, 2026 update with a new helpful guide and disk images to help new participants get started.



A comprehensive "Ultimate GlobalTalk Network Guide" (Google Doc format) covers the setup process end-to-end, including pre-made hard drive images. Questions about vintage Mac configuration are directed to TinkerDifferent.com.


**Source:** [MARCHintosh / GlobalTalk](https://marchintosh.com/globaltalk.html)

### Mini vMac + GlobalTalk Integration via TashTalk


Connecting an emulated Mini vMac instance to the GlobalTalk AppleTalk network is achievable using Mini vMac version 37 beta's LocalTalk over UDP (LToUDP) support, bridged through TashTalk.



The TashTalk 2 Hat is a Raspberry Pi hat that converts LocalTalk serial signals into multicast UDP packets picked up by LToUDP-enabled clients on the local network.



Once connected, a Mini vMac emulator can browse GlobalTalk AppleTalk zones from the Chooser, and participate in activities such as printing to an ImageWriter on another continent, chatting inside a HyperCard application, or playing Bolo with users in different countries.


**Source:** [vladovince.com — TashTalk + Mini vMac GlobalTalk guide](https://blog.vladovince.com/using-tashtalk-with-mini-vmac-to-participate-in-globaltalk-from-your-emulated-mac/)
