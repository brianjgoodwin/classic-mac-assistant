## Emulation

### QEMU 11.0.1 Point Release


QEMU released version 11.0.1 on May 25, 2026, alongside maintenance releases 10.2.3 and 10.0.10 on the same date.
 While QEMU is a general-purpose emulator, it remains one of the primary paths for running classic Mac OS 9 and Mac OS X in emulation, including via the community-maintained `qemu-system-ppc` and `qemu-system-m68k` targets. No Mac-specific changelog items were confirmed in this release at time of search.

**Source:** [QEMU Downloads](https://www.qemu.org/download/)

### DingusPPC Alpha 1.03: Disk Formatting and Full-Screen Mode


DingusPPC Alpha 1.03 added the ability to format disks using the hard disk setup program without third-party tools, and introduced a full-screen mode with zooming options. Alpha 1.04, released as a pre-release, added the ability to run Quake.
 
DingusPPC aims to emulate actual hardware rather than patching the ROM and RAM as SheepShaver does; the most functional machines are the PowerMac 6100 and PowerMac G3 Beige, and the emulator is under active development.


The 68kMLA forum thread for DingusPPC, started in January 2026, 
received a patch contribution addressing decrementer fixes, though interrupt issues remained under investigation.


**Source:** [DingusPPC Releases — GitHub](https://github.com/dingusdev/dingusppc/releases)

### E-Maculation Forums: Active Discussions on PPC Mac OS X Gaming Gap


An E-Maculation thread active through late May 2026 discussed the gap in emulation coverage for PPC and x86 Mac OS X games, which require an older version of Mac OS X to play.
 
Community members noted that for PPC, the Screamer build of QEMU-PPC is the primary option, while DingusPPC and MAME are described as "almost at the place" where they can run up to Mac OS X 10.2 with audio.


**Source:** [E-Maculation Forum](https://www.emaculation.com/forum/viewtopic.php?t=12411)

### Infinite Mac: Browser-Based Classic Mac Emulation Remains Active


The Infinite Mac project uses Basilisk II, SheepShaver, Mini vMac, DingusPPC, Previous, and PearPC as its underlying emulator backends, compiled to WebAssembly.
 
DingusPPC has been ported to WebAssembly and is accessible through Infinite Mac under "Custom: Experimental," capable of booting Mac OS 9.2.2 and early Mac OS X releases, though as of early 2025 the speed was noted as impractical for daily use in a browser context.
 The Infinite Mac GitHub repository (`mihaip/infinite-mac`) continues to receive commits; no major announcement was detected in the specific week window.

**Source:** [Infinite Mac — GitHub](https://github.com/mihaip/infinite-mac)

## Gaming

### *Return to Dark Castle* — Released on Steam and Mac App Store


*Dark Castle*, first released in 1986, returned in a newly remastered package for modern computers, bundling the original game with its sequel *Beyond Dark Castle* under the title *Return to Dark Castle*.
 
This 2026 Steam release is a remastered and expanded edition that combines levels from the original *Dark Castle* and *Beyond Dark Castle* with more than 50 brand-new rooms.
 
The game launched on macOS, Windows, and Valve's Steam Deck.



The remastering focused on clarity and feedback: color visuals help rooms read more cleanly, while remastered audio and preserved musical identity keep the game's personality intact, with the stated goal of making it playable on modern Macs without altering the core design.
 
The developer described this release as "the first step in reviving the Dark Castle series," with a new entry already in development.


**Source:** [Return to Dark Castle — Steam](https://store.steampowered.com/app/4054940/Return_to_Dark_Castle/) | [TinkerDifferent thread](https://tinkerdifferent.com/threads/dark-castle-is-back.4836/) | [Gamelevate coverage](https://gamelevate.com/return-to-dark-castle-is-out-now-on-steam-and-the-mac-app-store/)

### macintosh.world — New Community-Accessible Vintage Mac Portal (68kMLA, ~3 Days Ago)


A user on 68kMLA announced `macintosh.world`, a website described as an interactive "home page" for vintage computers that also functions on modern hardware; the project originated from replacing a hard drive on a PowerBook 180 with a Wi-Fi-enabled BlueSCSI.
 The 68kMLA thread is dated approximately 3 days before search time (i.e., late May 2026).


The site includes a live chat and direct messaging feature, and the creator described live chat between a 1992 PowerBook and a 2026 iPhone as functional.
 
A native client is available as a `.sit` archive, compiled in THINK C
 — meaning it runs directly on classic Mac hardware. 
The site includes a read-only Reddit browsing experience with images downscaled for low-bandwidth vintage hardware, accessible directly at `reddit.macintosh.world`.


**Source:** [68kMLA Thread — macintosh.world](https://68kmla.org/bb/threads/macintosh-world-new-website-for-old-macs.52393/)
