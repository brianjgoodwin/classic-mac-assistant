# Sources

The weekly tracker searches these categories of sources for vintage Apple computing content.

## Forums and Communities

- 68k Macintosh Liberation Army (68kMLA) forums
- E-Maculation forums (active emulation community)
- Macintosh Garden forums and discussion boards
- Vintage Computer Federation forums (Mac-specific sections)
- Mac GUI forum
- r/VintageApple (Reddit)
- r/ClassicMac (Reddit)
- r/RetroApple (Reddit)
- r/VintageComputing (Reddit - Mac-related threads)
- Classic Mac community on Mastodon
- AppleFritter forums
- TinkerDifferent forums (vintage Apple hardware)

## Project Hosting and Archives

- GitHub topics: `classic-mac`, `system-6`, `system-7`, `mac-os-classic`
- Macintosh Repository
- Macintosh Garden (software archive)
- Archive.org Classic Mac software collections
- Archive.org Internet Arcade (Mac games)
- Infinite Mac project updates
- DiscMaster collections (disk image archives)

## News and Blogs

- Low End Mac
- Vintage Mac Museum blog
- Classic Mac Networking
- Action Retro (YouTube/blog - Mac-related content)

## Emulation Communities

- Mini vMac forums and mailing lists
- Basilisk II / SheepShaver communities
- QEMU Mac emulation discussions

## Notes

The agent uses web search to find recent activity across these sources. This list
is a guide for the agent's search queries, not an exhaustive constraint. The agent
may find relevant content from other vintage Apple computing communities as well.
